# Placeholder Assets

These SVGs are **procedurally generated placeholders** — not final game art.

- Generated by `scripts/generate-placeholders.mjs` (run `npm run gen:assets`).
- Safe to delete and regenerate; output is deterministic.
- To use real art: replace any file here with your own image **keeping the same
  filename**, or point the `cover` paths in `src/data/siteData.ts` at new files.
- Raster art (PNG/JPG/WebP) works too — just update the path in `siteData.ts`.
